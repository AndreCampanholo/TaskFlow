# TaskFlow
Guia de instalação

## Pré-requisitos
Antes de começar, você precisa ter instalado:  
    - Node.js (versão 18 ou superior)  
    - Git

## Clonar o repositório
No terminal, execute os comandos:  
    `git clone https://github.com/AndreCampanholo/TaskFlow.git`  
    `cd frontend`

## Instalar dependências
Dentro da pasta do projeto, execute:  
    `npm install`

## Rodar o projeto
Para iniciar o projeto, execute:  
    `npx expo start`  
No computador, pressione w no terminal para abrir a aplicação no navegador.
No celular, instale o aplicativo Expo Go e escaneie o QR Code exibido no terminal.
O celular e o computador precisam estar conectados na mesma rede Wi-Fi.

## Variáveis de ambiente
O arquivo .env não vai para o GitHub, então é necessário criar antes de executar.
