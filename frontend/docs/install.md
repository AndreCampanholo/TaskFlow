# TaskFlow
Guia de instalação

## Pré-requisitos
Antes de começar, você precisa ter instalado:  
- Node.js (versão 18 ou superior)  
- Git


## Clonar o repositório
No terminal, execute os comandos:  
```bash
git clone https://github.com/AndreCampanholo/TaskFlow.git  
```


## Instalar dependências
Dentro da pasta do projeto, execute:  
```bash
npm install
```


## Rodar o projeto
Para iniciar o projeto, execute:  
    `npx expo start`  
### Navegador
Pressione `w` no terminal para abrir a aplicação no navegador.
### Dispositivo móvel
Instale o aplicativo `Expo Go` e escaneie o QR Code exibido no terminal.


## Variáveis de ambiente
O arquivo .env não vai para o GitHub, então é necessário criar antes de executar.
